## 🔧 Customization Options

### Add New Designs
```python
# In app/models.py or via admin panel
design = Design(
    title="Custom Design",
    category="Indian Mehndi",
    complexity="Medium",
    image_url="your-image-url"
)
db.session.add(design)
db.session.commit()
```

### Modify Colors
Edit `static/css/style.css` and change the CSS variables at the top:
```css
:root {
    --primary-color: #d4af37;  /* Gold */
    --secondary-color: #8b4789; /* Purple */
    /* ... more colors ... */
}
```

### Add New Features
1. Create models in `app/models.py`
2. Add routes in `app/routes.py`
3. Create templates in `templates/`
4. Add styling in `static/css/style.css`

### Common Issues

**Port 5000 Already in Use?**
```bash

# Windows
netstat -ano | findstr :5000
taskkill /PID <PID> /F
```

**Database Issues?**
```bash

python run.py
```

**Missing Dependencies?**
